# Common AGENTS.md Template

이 템플릿은 공통 규율을 위해 `mygit` 루트(예: `/Users/brandev/mygit/AGENTS.md`)에 배치하는 용도입니다.

## 기본 적용 범위

- 이 파일은 배치된 디렉터리와 그 하위 디렉터리 전체에 적용됩니다.
- 더 깊은 하위 폴더에 별도 `AGENTS.md`가 있으면 하위 파일이 상위 규칙을 우선합니다.
- 시스템/개발자/사용자 지침이 충돌할 경우 그쪽 지침이 우선합니다.

## 오픈AI 오더 정책

- 사용자 메시지에 OMx 트리거(`/prompts:*`, `$...`)가 있으면 해당 트리거를 우선 적용합니다.
- 트리거가 없으면 기본 모드는 `/prompts:executor`로 처리합니다.
- 다음 조건에서 기본 모드 전환을 수행합니다.
  - 아키텍처, 설계, 모듈 경계, 인터페이스/데이터 흐름 중심 요청: `/prompts:architect`
  - 일정/우선순위/범위/의존성/마일스톤 정리 요청: `/prompts:planner`

## 허용 스킬 우선순위(예시)

- 명시 시 최우선 적용: `$analyze`, `$autopilot`, `$build-fix`, `$code-review`, `$deepsearch`, `$security-review`, `$ultraqa`, `$ultrawork`, `$swarm`, `$plan`
- 기타 트리거는 `AGENTS` 기본 규칙이 없어도 일반 규칙에 준합니다.

## 실행 원칙

- 사용자의 의도를 빠르게 분해해, 가장 짧은 실행 단위로 진행합니다.
- 작업 중 애매하면 진행을 위해 확인 질문을 최소 한 번 제안합니다.
- 사용자 확인이 불필요한 범위에서는 빠르게 다음 액션으로 진행합니다.

## 공통 템플릿 적용 예시(복붙)

1. 기본값이 실행 우선순위에 맞지 않으면 `/prompts:executor`에서 `architect` 또는 `planner`로 분기.
2. 필요하면 `skill` 트리거를 사용자 요청에 맞게 즉시 재분기.
3. 불명확한 요구일 경우, 1단계 액션 제안 후 승인/수정 유도.
